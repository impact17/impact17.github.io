Style Guide

Explanations and demonstrations of styling for this website

Note: The corporate green colour is 015b63

This file is .../content/english/style.md

Since it’s at the top level, it can be accessed by appending /style in the URL bar. The idea is that this can be accessed by hand when needed; we won’t link to it.

It’s useful as it is displayed with the normal CSS, Javascript and so on for this site.

We created a very basic single page template at thems/airspace-hugo/layouts/_default/plain.html which basically just adds this content to the page.

In the front matter, we set layout: plain so Hugo will use our basic plain template.
